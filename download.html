Download MassOS

You can download MassOS now, to try it out and/or install it on your PC.

MassOS was discontinued in December 2022. The last stable release before discontinuation is MassOS 2022.10, which can be downloaded from the GitHub releases page (both Xfce and GNOME variants), alongside all previous versions. Note, however, that these legacy versions are outdated and may have security vulnerabilities, so are not recommended for use on real hardware (you should use a VM). Also be aware that newer hardware, particularly GPUs, may not be supported by builds with older versions of the Linux kernel.

As of January 2025, MassOS development has transitioned to an indefinite experimental phase. Any subsequent commits to the MassOS repository are done purely for testing purposes; they will not be given support and are not indicative of any future MassOS release. Nonetheless, these builds are made available from this web server. Before using these builds, it is strongly recommended that you read both the changelog and important notices in the changelog.md file in the MassOS GitHub repository. Also be aware that only Xfce builds are provided at this time.

After downloading, please read Installing MassOS for a detailed guide on how to create a bootable USB flash drive and install MassOS onto your computer.
